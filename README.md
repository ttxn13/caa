# ICGAS

## 一、运行依赖环境

| 序号 | 依赖项             | 补充项                     |
| ---- | ------------------ | -------------------------- |
| 1    | Visual Studio 2026 | 务必手动勾选MSVC143工具    |
| 2    | Qt 6.8.3           | 安装完需手动添加到系统path |
| 3    | Qt VS Tools        | 从VS扩展里面安装           |

## 二、自行安装项

1. AFSim及其端口配置

| 类型        | IP             | Port |
| ----------- | -------------- | ---- |
| AFSim客户端 | 192.168.50.255 | 8888 |
| AFSim服务器 | 192.168.50.255 | 9999 |

2. Tacview及其端口配置

| 序号 | 名称 | 内容      |
| ---- | ---- | --------- |
| 1    | IP   | localhost |
| 2    | Port | 10009     |
| 2    | user | ICGAS     |
| 3    | pass | `空密码`  |
